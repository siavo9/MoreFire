# Stop The Fire! 🚒🔥💧

A simple, fast-paced firefighter tap game. Tap **Play Now** and a random scene erupts in flames — put out every fire before the 30-second timer runs out to advance. Level 1 has 5 fires, level 2 has 10, level 3 has 15, and so on. How far can you go?

## Run locally

```
npm install
npm start
```

Open http://localhost:3000

## Deploy

Deployed on Vercel from the `main` branch.
